# Doomgeneric Raylib

A simple port of Doom to Raylib 4.0 using DoomGeneric.

# Building

Clone the repository and the submodules:
```
git submodule update --init --recursive
```
Build raylib:
```
cd src/raylib/src; make -j
```
And build the game with `make`.
